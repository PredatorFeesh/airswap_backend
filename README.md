# AirSwAP Backend

## To run

Make sure you have `venv` installed. Then create your virtual env:

```bash
python3 -m venv venv
```

Now, activate via:

On Linux and Mac:
```bash
./venv/bin/activate
```

On Windows:
```bash
source venv\Scripts\activate.bat
```

Now install

```bash
pip3 install -r requirements.txt
```

Now you can run via `python3 App.py`
